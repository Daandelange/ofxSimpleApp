# ofxSimpleApp

A multi-purpose application layout for OpenFrameworks projects, powered by ofxImGui.

Features :
	- Minimal by default, fine-tuneable using a simple opt-in based config file.
	- A basic application layout with a menu, some common keybindings and metrics to get your app going quickly.
	- A rendering FBO and basic window setup.
	- An in-app logs console to keep an eye on the internal meccanics on runtime.
	- Debugging features 
	- Easy setup of various addons :
		- ofxSyphon for Syphon input and output.
